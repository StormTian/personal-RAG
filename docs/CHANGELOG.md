### 新增
- 初始版本发布
- 实现基础 RAG 功能

## 版本说明

### 版本号格式

版本格式：`主版本号.次版本号.修订号`

1. **主版本号**：不兼容的 API 修改
2. **次版本号**：向下兼容的功能新增
3. **修订号**：向下兼容的问题修复

### 版本阶段

- **Alpha**: 内测版本，功能不完整
- **Beta**: 公测版本，功能基本完整
- **RC** (Release Candidate): 候选版本，准备发布
- **Release**: 正式发布版本

### 当前版本状态

**状态**: Stable
**最新版本**: 0.3.0
**维护周期**: 活跃维护中

## 升级指南

### 从 0.2.x 升级到 0.3.x

1. 备份文档库
2. 更新代码
   ```bash
   git pull origin main
   ```
3. 删除旧缓存
   ```bash
   rm .index_cache.pkl
   ```
4. 重新运行
   ```bash
   python web_app.py
   ```

### 从 0.1.x 升级到 0.2.x

1. 检查配置文件
2. 更新依赖
   ```bash
   pip install -r requirements.txt
   ```
3. 重建索引
   ```bash
   rm .index_cache.pkl
   python app.py --list-docs
   ```
